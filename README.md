# Solomon Portfolio | Industrial Minimalist Engineering

> **Strategic Technology Partner & Fractional CTO**  
> _High-performance business platforms engineered with technical precision._

This repository hosts the professional architecture of a Fractional CTO's portfolio, built as a demonstration of high-level web engineering and strategic design. It prioritizes Core Web Vitals, type-safety, and modular industrial aesthetics.

## 🛠 Strategic Stack

- **Core Framework**: [Next.js 15](https://nextjs.org/) (App Router, Server Components)
- **Engineered Styling**: [Tailwind CSS 4](https://tailwindcss.com/)
- **Logic**: [TypeScript](https://www.typescriptlang.org/) (Strict Mode)
- **Motion & Dynamics**: [Framer Motion](https://www.framer.com/motion/)
- **Infrastructure Context**: CCNA-aligned network architectural principles

## 📐 Design Philosophy: Industrial Minimalism

The interface is designed with a focus on:
- **High-Contrast Grids**: Technical clarity and structural integrity.
- **Micro-Animations**: Purposeful transitions that enhance UX without bloat.
- **Modular UI Patterns**: Reusable, atomic components designed for scale.
- **Zero Placeholder Policy**: Real-world assets and engineered solutions only.

## 🚀 Performance Metrics

Optimized for 95+ scores across all Core Web Vitals:
- **LCP (Largest Contentful Paint)**: < 1.2s
- **FID (First Input Delay)**: < 100ms
- **CLS (Cumulative Layout Shift)**: < 0.1

## 📦 Deployment & Setup

### Requirements
- Node.js 20+
- npm / pnpm / bun

### Local Development
```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Production build
npm run build
```
